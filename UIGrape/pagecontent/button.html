Button Component


Import

    import { Button } from 'ui-grape';
    import { faBell } from '@fortawesome/free-solid-svg-icons';

Code Example

The Button component allows you to create simple Buttons. Here's an example of how you can use it:

      <Button 
        children= 'Ui Grape Button'
        rightIcon= {faBell}
        type= 'danger'
        size= 'small'
      />
