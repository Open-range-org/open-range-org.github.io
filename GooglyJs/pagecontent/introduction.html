Introduction

Googly.js is a lightweight JavaScript validation library. If you’re unfamiliar with handling validation in JavaScript or if you’re looking for a library to assist with validation, Googly.js is a great choice.

Installation

To use this library, which relies on jQuery as its base JavaScript library, follow these steps: Include jQuery: Make sure you add jQuery to your project. Download “googly.js”: Obtain the “googly.js” file from either GitHub or CDN. You can find it there and include it in your project. By following these steps, you’ll have both jQuery and the “googly.js” library available for your web application. 
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
      <script src="./dist/googly.min.js"></script>
